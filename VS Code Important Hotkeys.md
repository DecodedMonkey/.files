### Turbo Console Log (Extention):

|  Name | Hotkey  |
| ------------ | ------------ |
| Create Rocket Log | `Shift` + `Alt` + `L`  |
| Comment All logs Messages  |  `Shift` + `Alt` + `C`  |
| Delete All Rocket logs  |  `Shift` + `Alt` + `D` |
| Uncomment All Rocket Logs  | `Shift` + `Alt` + `U`  |

### Bookmarks (Extention):
